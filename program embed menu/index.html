      Séléctionnez la section correspondant au widget que vous voulez intégrer.

    • 3.
      Visualisez les modification dans l'aperçu du widget.
    • 4.
      Une fois satisfait, sauvegardez vos modifications et récupérez le code pour le placer dans votre site.

    • Personnalisation du css

    • Saisissez votre css
      Structure css de la liste:
      • .list-items: canevas de la liste d'événements
        • .plis: tete de section
        • .pli: bloc événement avec article
          • .ei: bloc événement
            • .evpic: canevas de l'image
            • .evcnt: canvevas des descriptifs
              • .evtitle: titre d'un événement
              • .evdesc: description courte d'un événement
              • .evspacetime: lieu et dates d'un événement
              • .ecat: catégorie
      Structure css d'un événement:
      • #event: canevas de l'événement
        • .main: informations principales de l'événement
          • .language-list: séléction des langues
          • #description: section des descriptifs
            • h1: titre
            • .short: description courte
            • .spacetime: info générale sur les lieux et horaires
            • .tags: mots clés
            • .free: description longue
          • #places: section des lieux
            • .section-head: titre de la section
            • .place-detail: addresse d'un lieu séléctionné
            • .map: carte
          • #dates: section des dates et horaires
            • .section-head: titre de la section
            • .months: mois (si il y a plus de 10 dates)
              • li: un mois (si passé, contient la classe '.passed'
                • a: intitule du mois
                • span: nombre de dates dans le mois
            • .dates: dates
              • .date: intitule de la date
              • ul li: horaires
        • .sec: informations secondaires
          • .poster: canvevas de l'affiche
