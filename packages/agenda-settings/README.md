# Agenda-settings

Voici les fameuse applications react permettant de créer un agenda, de modifier ses paramètres et malheureusement de le supprimer.

## Installation

1. Copiez le fichier testconfig.sample.js vers testconfig.js et modifiez les données si necessaire
2. Faites `npm i`
3. C'est tout

## Test de l'application

Pour dérouler les tests unitaires et fonctionnels faites `npm start` et prenez votre souris !

L'application de test se lance sur localhost:3000 pour l'app de création et localhost:3000/?_app=edition pour l'app d'édition
